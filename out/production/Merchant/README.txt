Проект Merchant
-----------------

В проекте производится расчет
заработной платы продавца, в зависимости
от объема продаж, по прогрессивной шкале.

Проект демонстрирует:
- конструкторы,
- методы,
- наследование,
- интерфейсы.


Задание
----------
1) Создать клон проекта Merchant, например, Merchant-01.
2) Разработку этого клона вести под Git и на GitHub.
3) В проекте-клоне должны быть ветки master и develop.
4) В ветке develop реализовать ввод данных через Scanner.
5) При разработке отработать: коммиты, ветвление, слияние.
6) Слить ветку develop с веткой master, оставив приоритетом
ввод через Scanner. После слияния с веткой master, ветку develop
не удалять.
7) После завершения проекта в LMS указать ссылку на GitHub.
